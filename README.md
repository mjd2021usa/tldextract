# tldextract

TLD Extract written in Go.  Extracts TLD, domain, and subdomains from a host or URL 

Installation
==========
Install tldextract:
```sh
go get github.com/mjd2021usa/tldextract

```

Examples:

Simple example:
```sh
  examples/01/main.go
```

Integration Test example:
```sh
  examples/02/main.go
```
